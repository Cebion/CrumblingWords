Crumbling Words by ThunderPerfectWitchcraft, 2023

Start the game by kicking CrumblingWords.exe on Windows, or by using
./CrumblingWords.x86_64 when using Linux (needs SDL2, SDL2mixer, SDL2image and SDL2ttf libs).

Use the mouse to skip the intro, to use the menus and to play the game.
Consult the ingame help to learn about the gameplay and possible settings.

If you encounter problems or questions, contact us
contact@thunderperfectwitchcraft.org

Crumbling Words is free libre open source software (FLOSS).
See the license.txt for further details or HowToCompile.txt for build instructions.

Have fun!
